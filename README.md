# D-Day Pixel Wall

'D-Day 픽셀 월'은 여러 사용자가 자신의 D-Day를 거대한 디지털 캔버스에 함께 공유하는 웹 애플리케이션입니다. 개인의 소중한 D-Day 경험을 다른 사람들과 함께 응원하며 독특한 사회적 경험을 나누는 것을 목표로 합니다.

## ✨ 주요 기능



- **다국어 지원**: 브라우저 언어(한국어/영어)에 따라 UI가 자동으로 변경됩니다.

- **실시간 공유 캔버스**: 다른 사용자가 추가하거나 '좋아요'한 타일이 10초 간격으로 자동 갱신됩니다.

- **D-Day 등록**: 캔버스의 빈 공간을 클릭하여 나만의 D-Day 타일 등록

- **소셜 공유**: 타일의 고유 주소를 복사하여 다른 사람에게 공유할 수 있으며, 소셜 미디어에 공유 시 타일 정보가 담긴 미리보기가 표시됩니다.

- **캔버스 탐색**: 마우스 드래그를 통한 이동(Pan) 및 휠을 통한 확대/축소(Zoom) 지원

- **반짝임 효과**: D-Day 당일(`D-0`)이 된 타일은 반짝이는 애니메이션 효과 적용

- **자동 소멸**: D-Day가 7일 이상 지난 타일은 매일 자동으로 삭제



## 🛠️ 기술 스택



- **Framework**: SvelteKit (Svelte + Vite)

- **Language**: TypeScript

- **Database**: SQLite

- **ORM**: Prisma

- **Canvas**: Konva.js

- **Styling**: TailwindCSS



## 🚀 시작하기



### 1. 의존성 설치



프로젝트 디렉토리에서 아래 명령어를 실행하여 필요한 패키지를 설치합니다.



```bash

npm install

```



### 2. 데이터베이스 설정



Prisma를 사용하여 SQLite 데이터베이스와 테이블을 생성합니다. `.env` 파일이 없다면, `DATABASE_URL="file:./prisma/dev.db"` 내용으로 생성해주세요.



```bash

npx prisma migrate dev

```



### 3. 개발 서버 실행



아래 명령어를 실행하여 개발 서버를 시작합니다.



```bash

npm run dev

```



서버가 시작되면 터미널에 표시되는 주소(예: `http://localhost:5173`)로 접속하여 애플리케이션을 확인할 수 있습니다.



## 📝 최근 변경사항



- **다국어(i18n) 지원**: `svelte-i18n` 라이브러리를 도입하여 한국어/영어를 지원합니다. 사용자의 브라우저 언어에 따라 UI가 자동으로 변경됩니다.

- **실시간 타일 동기화**: 10초마다 변경된 타일 정보를 자동으로 가져와 화면을 갱신하는 기능을 추가하여 사용자 경험을 개선했습니다.

- **소셜 공유(OG 태그) 기능**: 타일 공유 시 소셜 미디어에 타일의 제목, D-Day, 좋아요 수가 포함된 미리보기가 표시되도록 OG 태그를 동적으로 생성하는 기능을 추가했습니다.

- **UI/UX 개선**: 로고 클릭 시 서비스 소개 모달이 표시되도록 하고, 전체적인 UI 텍스트와 스타일을 개선했습니다.

- **기능 오류 수정**: '좋아요' 기능, Svelte 컴파일, Prisma 클라이언트 초기화 관련 런타임 오류를 수정하여 안정성을 높였습니다.
